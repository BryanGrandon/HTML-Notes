# [Table label](/code/table.html)

Tables have been included in HTML since its earliest versions and are a fantastic way to display data clearly.

```HTML
<!-- Create table -->
<table>
  <!-- Table title -->
  <caption>Title for table</caption>
  <!-- Table header -->
  <thead>
    <!-- Table row -->
    <tr>
      <!-- Table head -->
      <th>Element</th>
    </tr>
  </thead>
  <!-- Table body -->
  <tbody>
    <tr>
      <!-- Table data -->
      <td>Element</td>
    </tr>
  </tbody>
  <!-- Table footer -->
  <tfoot>
    <tr>
      <td>Element</td>
    </tr>
  </tfoot>
</table>
```
